# Practica Centralita - POO en C#

Este proyecto implementa una central telefónica aplicando los 4 pilares de la Programación Orientada a Objetos:

## Pilares aplicados

- **Abstracción:** Clase abstracta `Llamada`
- **Encapsulamiento:** Atributos privados con propiedades
- **Herencia:** Clases `LlamadaLocal` y `LlamadaProvincial`
- **Polimorfismo:** Uso de `override` en métodos como `CalcularPrecio()`

## Funcionalidades

- Registro de llamadas
- Cálculo de costos
- Diferentes tipos de llamadas (locales y provinciales)
- Resumen total de llamadas y facturación

## Archivos

- Llamada.cs
- LlamadaLocal.cs
- LlamadaProvincial.cs
- Centralita.cs
- Program.cs
